# Szekely Guidelines Project

**Version 0.0.1**

A collection of tools for working with the United States Sentencing Guidelines Manual (2025 edition).

## Overview

This project provides web-based tools to assist federal criminal defense attorneys with sentencing calculations and guideline analysis.

## Current Features

### Guidelines Viewer
A simple PDF viewer that allows you to quickly access any page of the 2025 U.S. Sentencing Guidelines Manual.

**Usage:**
1. Open `guideline-viewer.html` in your web browser
2. Enter a page number (1-553)
3. Press Enter or click "View" to display the PDF
